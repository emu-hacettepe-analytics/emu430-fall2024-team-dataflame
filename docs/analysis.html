Analysis of Forest Fires in Turkey

This page presents the exploratory data analysis of forest fire statistics in Turkey between 1988 and 2024.

Key Takeaways

  • Forest fires in Turkey fluctuate significantly over time.
  • Some years experienced extremely high burned forest areas.
  • Burned areas and fire frequency do not always increase together.
  • Negligence-related fires constitute a major proportion of forest fire causes.
  • Forest fires remain an important environmental issue in Turkey.

Fire Causes Analysis

Code 
causes <- fires %>%
  summarise(
    Intentional = sum(intentional_number, na.rm = TRUE),
    Negligence = sum(negligence_number, na.rm = TRUE),
    Natural = sum(natural_number, na.rm = TRUE),
    Unknown = sum(unknown_number, na.rm = TRUE)
  ) %>%
  pivot_longer(
    cols = everything(),
    names_to = "Cause",
    values_to = "Count"
  )

ggplot(causes, aes(x = reorder(Cause, Count), y = Count, fill = Cause)) +
  geom_col() +
  coord_flip() +
  labs(
    title = "Total Forest Fires by Cause",
    x = "Cause",
    y = "Total Number of Fires"
  ) +
  theme_minimal()

Negligence-related fires represent one of the largest categories among forest fire causes.

Burned Area by Fire Cause

hectares <- fires %>%
  summarise(
    Intentional = sum(intentional_hectare, na.rm = TRUE),
    Negligence = sum(negligence_hectare, na.rm = TRUE),
    Natural = sum(natural_hecatre, na.rm = TRUE),

  pivot_longer(
    cols = everything(),

  coord_flip() +
  labs(
    title = "Burned Area by Fire Cause",
    x = "Cause",
    y = "Burned Area"
  ) +
  theme_minimal()

Negligence-related fires account for a substantial proportion of total burned forest areas.

Top 10 Years with Highest Burned Area

fires %>%

  slice(1:10) %>%
  ggplot(aes(x = reorder(as.factor(year), burned_area), y = burned_area)) +
  geom_col(fill = "firebrick") +
  coord_flip() +
  labs(
    title = "Top 10 Years with Highest Burned Area",
    x = "Year",
    y = "Burned Area"
  ) +
  theme_minimal()

2021 appears as the most destructive year in terms of burned forest area.

Conclusion

Forest fires continue to be an important environmental issue in Turkey. The analysis shows that both fire frequency and burned area vary considerably over time. Human-related causes, especially negligence, play an important role in forest fire occurrence.
